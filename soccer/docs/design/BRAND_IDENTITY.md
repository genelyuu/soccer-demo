> 근거: soccer_rnd REFERENCES.md (30개 학술 문헌), METRICS_FORMULAS.md (스포츠 부하/회복 지표)
> 핵심 키워드: **건강(Health)**, **안전(Safety)**, **스포츠(Sports)**, **전문성(Expertise)**, **신뢰성(Reliability)**

## 1. 브랜드 철학

### 미션
> "모든 선수가 **건강**하게 뛰고, **안전**하게 돌아오는 **스포츠** 환경을 **전문**적으로 운영하고, **신뢰**할 수 있는 기록으로 남긴다."

### 근거: soccer_rnd 도메인에서의 발견

| 학술 근거 | 핵심 메시지 | BI 반영 |
|-----------|-----------|---------|
| Gabbett (2016) [Ref #9] "훈련-부상 역설" | 적절한 부하가 오히려 부상을 예방한다 | **안전**: 무리하지 않는 일정 관리 |
| Hooper & Mackinnon (1995) [Ref #18] 과훈련 모니터링 | 피로/스트레스/수면/근육통이 건강의 지표 | **건강**: 선수 컨디션 중심 운영 |
| Foster (1998) [Ref #1] 훈련 모니터링 | 체계적 기록이 과훈련을 예방한다 | **스포츠**: 데이터 기반 운영 루프 |
| Bourdon et al. (2017) [Ref #27] 부하 모니터링 합의문 | 주관적 지표(RPE)가 객관적 지표보다 효과적 | **건강**: 선수 자기보고 중심 |
| Saw et al. (2016) [Ref #21] 주관적 모니터링 | 자기 보고 측정이 가장 신뢰할 수 있다 | **신뢰성**: 출석/컨디션 자기 투표 |
| Task Force ESC (1996) [Ref #22] HRV 표준 | 국제 표준 기반 측정이 전문성의 근거 | **전문성**: 학술 기반 지표 체계 |
| Impellizzeri et al. (2019) [Ref #30] 내적/외적 부하 개념 정리 | 15년간 축적된 개념 체계가 신뢰성 보장 | **신뢰성**: 검증된 프레임워크 적용 |

### 브랜드 가치 오각형

```
           건강 (Health)
          /      |      \
         /       |       \
  전문성 /   PITCH FC  \ 신뢰성
(Expertise)     |     (Reliability)
         \       |       /
          \      |      /
  안전 (Safety)——+——스포츠 (Sports)
```

- **건강**: 선수 컨디션을 최우선으로. 무리한 일정보다 적절한 휴식.
- **안전**: 체계적 출석 관리로 부상 위험 감소. 기록을 통한 패턴 파악.
- **스포츠**: 즐거운 경기, 공정한 기록, 팀워크 중심.
- **전문성**: 스포츠 사이언스(ATL/CTL/ACWR, HRV) 기반 운영 체계. 학술 문헌에 근거한 지표 설계.
- **신뢰성**: 기록의 무결성, 역할 기반 접근 제어(RBAC), 변경 이력 추적. 데이터가 곧 신뢰.

## 5. 핵심 캐릭터: **피치(Pitchy)**

### 캐릭터 프로필

| 항목 | 값 |
|------|---|
| 이름 | **피치(Pitchy)** |
| 종류 | 의인화된 축구공 |
| 성격 | 쾌활하고 책임감 있는 **전문** 팀 매니저 |
| 슬로건 | "오늘도 건강하게 한 판! 기록은 내가 **신뢰**할 수 있게." |
| 전문성 표현 | 안경 착용 (데이터 분석가), 클립보드에 ACWR 그래프 |
| 신뢰성 표현 | 가슴에 체크마크 배지 (검증된 기록), 일관된 미소 |

### 외형 설명 (ASCII 디자인 보드)

```
        ___________
       / ⌐●   ●¬  \        ← 눈: 둥근 뿔테 안경 (전문성) + 밝은 눈 (건강한 에너지)
      |    ◡        |       ← 입: 미소 (긍정, 친근, 일관된 신뢰)
      | [✅]        |       ← 가슴: 체크마크 배지 (신뢰성)
      |  ⬡  ⬡  ⬡   |       ← 몸: 축구공 오각형 패턴 (primary 컬러)
       \ ⬡  ⬡  ⬡  /
        \_________/
         |  👟  |            ← 발: 축구화 (스포츠)
        /|      |\
       / |      | \
      💚  🩹    📋📈         ← 소품: 하트(건강), 반창고(안전), 클립보드+그래프(전문성/기록)
```

### 캐릭터 활용 시나리오

| 상태 | 피치의 표정/행동 | 메시지 |
|------|----------------|--------|
| 빈 상태 (경기 없음) | 축구공을 들고 기다리는 포즈 | "경기를 만들어볼까요?" |
| 투표 진행 중 | 체크리스트를 들고 손 흔드는 포즈 | "투표 잊지 마세요!" |
| 경기 확정 | 호루라기를 부는 포즈 | "준비 완료! 건강하게 뛰어요!" |
| 기록 완료 | 트로피를 들고 기뻐하는 포즈 | "수고했어요! 기록이 저장됐어요." |
| 에러 발생 | 당황한 표정 + 반창고 | "앗, 문제가 생겼어요. 다시 시도해볼까요?" |
| 팀 없음 | 혼자 앉아있는 포즈 | "팀에 합류하고 함께 뛰어요!" |

### 캐릭터 디자인 원칙

1. **둥근 형태**: 축구공 기반, 각진 요소 최소화 → 친근함
2. **primary 컬러 기반**: 몸체는 `#2B9E6B` 녹색 + 흰색 오각형 패턴
3. **소품 5가지**: 💚 하트(건강), 🩹 반창고(안전), 📋 클립보드(스포츠), 🔬 렌즈(전문성), ✅ 배지(신뢰성)
4. **안경**: 둥근 뿔테 안경 — 데이터를 분석하는 전문가 이미지
5. **표정 변화**: 상태별 6가지 표정으로 앱 전체에서 일관된 가이드 역할
5. **사이즈**: 빈 상태 일러스트에 사용 (64x64 ~ 128x128)
