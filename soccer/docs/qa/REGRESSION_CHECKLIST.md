# 회귀 체크리스트 (10분) (Task 0020)

> 수동 테스트 가이드. 주요 변경 후 10분 내에 수행.

## 1. 인증 (2분)
- [ ] 회원가입 성공 (이메일/비밀번호/이름)
- [ ] 로그인 성공
- [ ] 로그인 후 세션 유지 (새로고침 후에도 로그인 상태)
- [ ] 잘못된 비밀번호로 로그인 실패

## 2. 경기 관리 (3분)
- [ ] 경기 목록 조회 (/matches)
- [ ] 새 경기 생성 → 목록에 표시
- [ ] 경기 상세 조회 (/matches/[id])
- [ ] 출석 투표 현황 표시 (전원 PENDING 초기값)

## 3. 출석/투표 (2분)
- [ ] 참석 투표 → 상태 변경 확인
- [ ] 불참 투표 → 상태 변경 확인
- [ ] 투표 변경 (참석 → 불참) → 반영 확인

## 4. 경기 확정 + 기록실 (2분)
- [ ] 경기 확정 → 상태 CONFIRMED 변경
- [ ] 기록실 자동 생성 확인
- [ ] 기록 입력 (득점 1) → 테이블 반영
- [ ] 기록 수정 → 업데이트 반영

## 5. 팀 관리 (1분)
- [ ] 내 팀 목록 조회 (/team)
- [ ] 멤버 목록 조회 (/team/members)

---

**총 예상 시간**: 10분
**실패 시**: 해당 영역의 E2E 시나리오로 상세 재현
