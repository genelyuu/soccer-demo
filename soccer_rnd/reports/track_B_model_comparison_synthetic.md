## 1. 분석 목적

본 보고서는 **훈련 부하 구조 지표(ACWR, Monotony)가 다음날 Hooper Index를 얼마나 잘 예측하는지** 통계적으로 평가하는 파이프라인의 정합성을 검증하기 위해, 알려진 참값 파라미터로 생성된 합성 데이터에 대해 네 가지 모형을 비교한다.

핵심 질문:

1. 모형이 합성 데이터의 참값 계수(beta_acwr = 2.5, beta_monotony = 1.5)를 적절히 복원하는가?
2. OLS에서 혼합효과모형으로의 확장이 적합도를 향상시키는가?
3. Monotony를 추가 예측변수로 투입하면 유의한 추가 설명력이 있는가?
4. Rolling ACWR과 EWMA ACWR 중 어느 산출 방식이 더 적합한가?

본 분석은 **파이프라인 검증 목적**으로 수행되었으며, 합성 데이터의 결과를 실제 스포츠과학적 현상으로 직접 해석할 수 없다.

## 2. 합성 데이터 설계

### 생성 모델

```
hooper_next(t) = beta_0 + beta_acwr * acwr_rolling(t) + beta_monotony * monotony(t) + u_j + epsilon(t)
```

|----------|-----|------|
| 선수 수 | 12명 | athlete_id: A01–A12 |
| 관측 기간 | 120일 | daily_load 기반 |
| 분석 관측수 | 1,116 | 워밍업 기간(27일) 제거 후 |
| daily_load | 평균 400 AU, SD 80 | 주말(토/일) 0.5배 |
| beta_0 (절편) | 12.0 | 기저 Hooper Index |

| sigma_noise | 1.8 | 잔차 노이즈 SD |

## 3. 모형 설계

| 모형 | 유형 | 고정효과 | 랜덤효과 | 추정 방법 |
|------|------|----------|----------|-----------|
| M1 | OLS | ACWR (Rolling) | -- | 최소자승법 |
| M2 | Mixed Effects (LMM) | ACWR (Rolling) | (1\|athlete) | ML |
| M3 | Mixed Effects (LMM) | ACWR (Rolling) + Monotony | (1\|athlete) | ML |
| M4 | Mixed Effects (LMM) | ACWR (EWMA) + Monotony | (1\|athlete) | ML |

### 모형 선택 근거

- **M1 → M2**: 선수 간 기저 Hooper 수준 차이를 무시하면 잔차 분산이 과대 추정되고, 고정효과 추정에 편향이 발생할 수 있다. 랜덤 절편으로 이를 해소한다.
- **M2 → M3**: 부하의 "양"(ACWR)뿐 아니라 "패턴"(Monotony)이 피로에 기여하는지 검증한다. 단조로운 부하 패턴은 심리적/생리적 적응 실패와 관련될 수 있다 (Foster, 1998).
- **M3 → M4**: Rolling ACWR 대신 EWMA ACWR을 투입하여 산출 방식 간 차이를 비교한다. EWMA는 최근 부하에 더 높은 가중치를 부여한다 (Williams et al., 2017).

### AIC/BIC 산출

모든 혼합효과모형은 **ML(Maximum Likelihood) 추정**으로 적합하였다. 고정효과 구조가 다른 모형 간 비교에서 ML 기반 AIC/BIC가 적절하다 (Zuur et al., 2009).

## 5. 결과 해석

### 5.1 계수 방향 확인 (참값과 비교)

**Monotony 계수**: M3에서 1.4727, M4에서 1.4555로 추정되어, 참값(1.5)에 매우 근접하게 복원되었다. 이는 Monotony의 효과가 충분한 분산(SD = 0.540)을 가지므로 안정적으로 추정될 수 있음을 시사한다.

**ACWR 계수**: M3에서 1.9376으로 추정되어 참값(2.5)보다 과소추정되었다. 이는 ACWR의 분산(SD = 0.065)이 매우 작아 기울기 추정의 불확실성이 높기 때문이다. 표준오차(0.867)가 계수 자체(1.938)의 약 45%에 달하며, 이는 소표본과 낮은 변동성이 결합된 효과이다.

**M1/M2에서 ACWR 계수가 유의하지 않은 이유**: Monotony가 모형에 포함되지 않은 M1(coef = 0.678, p = 0.584)과 M2(coef = 0.567, p = 0.543)에서는 ACWR이 유의하지 않았다. 이는 합성 데이터에서 ACWR과 Monotony가 모두 Hooper에 영향을 미치도록 설계되었으나, ACWR 단독으로는 Monotony의 효과를 설명하지 못하여 잔차 분산이 커지고, ACWR 변동 범위가 좁아(SD = 0.065) 신호 대 잡음비가 낮기 때문이다. Monotony를 추가한 M3에서야 ACWR이 유의해지는(p = 0.025) 것은 누락변수 편향(omitted variable bias)이 해소된 결과로 해석된다.

### 5.2 혼합효과모형 vs OLS (M1 → M2)

M1(OLS)에서 M2(Mixed)로의 전환에서 극적인 개선이 관찰되었다:

- **AIC**: 5,365.47 → 4,779.39 (delta = -586)
- **R²**: 0.0003 → 0.4434
- **RMSE**: 2.6729 → 1.9944 (25.4% 감소)

OLS에서 R²가 사실상 0인 것은, 선수 간 기저 Hooper 수준의 대폭적 변동(sigma_athlete = 1.5)이 잔차에 혼입되어 ACWR의 미세한 효과를 완전히 마스킹하기 때문이다. 랜덤 절편이 이 변동을 흡수하면서 R²가 0.443으로 급상승하였다. 이는 반복 측정 데이터에서 개인차를 무시하는 OLS의 근본적 한계를 잘 보여준다.

랜덤 절편 분산(3.124)은 참값(2.25)보다 높게 추정되었는데, 이는 Monotony 미포함으로 인해 선수별 Monotony 평균 차이가 랜덤 절편에 일부 흡수되었기 때문으로 해석된다.

### 5.3 Monotony 추가 효과 (M2 → M3)

Monotony를 추가한 M3은 M2 대비 모든 지표에서 유의미하게 개선되었다:

- **AIC**: 4,779.39 → 4,603.90 (delta = -175.5)
- **R²**: 0.4434 → 0.5260 (delta = +0.083)
- **RMSE**: 1.9944 → 1.8405 (7.7% 감소)
- **Monotony 계수**: 1.4727 (p < 0.001) — 참값(1.5)에 매우 근접

AIC 감소폭(-175.5)은 매우 크며, Monotony가 ACWR과 독립적으로 Hooper Index 변동의 상당 부분을 설명함을 강력히 시사한다. 이는 합성 데이터 생성 시 Monotony를 예측변수로 포함한 것과 일치한다.

또한, M2에서 유의하지 않았던 ACWR 계수(0.567, p = 0.543)가 M3에서는 유의해졌다(1.938, p = 0.025). 이는 Monotony가 공변량으로 포함됨으로써 잔차 분산이 감소하고, ACWR의 독립적 효과가 드러난 것으로 해석된다.

### 5.4 Rolling vs EWMA (M3 vs M4)

| 비교 항목 | M3: Rolling | M4: EWMA | 차이 |
|-----------|:-----------:|:--------:|:----:|
| AIC | **4,603.90** | 4,604.22 | +0.32 |
| BIC | **4,628.99** | 4,629.31 | +0.32 |
| R² | **0.5260** | 0.5259 | -0.0001 |
| RMSE | **1.8405** | 1.8407 | +0.0002 |

Rolling ACWR 기반 모형(M3)이 EWMA 기반 모형(M4)보다 근소하게 우수하였다. AIC 차이(0.32)는 실질적으로 무의미한 수준이며, 두 산출 방식은 사실상 동등한 적합도를 보였다. 합성 데이터가 Rolling ACWR로 생성되었음에도 차이가 미미한 것은, 두 ACWR 산출 방식 간 상관이 매우 높기 때문이다. 실제 데이터에서는 부하 변동 패턴에 따라 두 방식의 차이가 더 명확해질 수 있다.

## 6. 실제 데이터와의 비교 논의

### 실제 데이터 분석 결과 요약 (Track B: SoccerMon)

| 항목 | 실제 데이터 | 합성 데이터 |
|------|:-----------:|:-----------:|
| 선수 수 | 44명 | 12명 |
| 관측수 | 16,186 | 1,116 |
| M1 R² (OLS) | 0.000 | 0.0003 |
| M2+ R² (Mixed) | 0.453~0.454 | 0.443~0.526 |
| ACWR 계수 (M4) | **-0.078** (p < 0.001) | +1.644 (p = 0.031) |
| Monotony 계수 (M4) | **+0.142** (p = 0.002) | +1.456 (p < 0.001) |
| AIC (M1 → M2) | 64,509 → 54,751 | 5,365 → 4,779 |

### 주요 차이점 서술

1. **ACWR 계수 방향의 차이**: 합성 데이터에서 ACWR 계수는 양의 방향(+1.938)으로 참값(+2.5)과 일치하였으나, 실제 데이터에서는 **음의 방향(-0.078)**으로 나타났다. 실제 데이터의 음의 ACWR 계수는 직관적 예상(ACWR 증가 → 피로 증가 → Hooper 증가)과 반대 방향으로, 이는 ACWR 증가가 훈련 적응의 결과이거나 체력 상태가 좋을 때 부하를 높이는 선택 편향이 작용할 수 있음을 시사한다. 합성 데이터에서는 이러한 역인과 관계가 반영되지 않았다.

2. **OLS에서 혼합효과로의 전환 효과**: 양쪽 모두 OLS R²가 사실상 0이었으며, 혼합효과모형으로 전환 시 R²가 0.44~0.53으로 급상승하였다. 이는 개인차가 지배적인 반복 측정 데이터에서 랜덤 절편의 결정적 중요성을 일관되게 보여준다.

3. **Monotony의 역할 차이**: 합성 데이터에서 Monotony는 강력한 예측변수(coef = 1.473, p < 0.001)였으나, 실제 데이터에서는 Strain 없이 투입 시 비유의적(coef = -0.027, p = 0.191)이었다가 Strain 투입 후 유의해졌다(coef = +0.142, p = 0.002). 실제 데이터에서의 억제변수 효과(suppressor effect)는 합성 데이터에서 재현되지 않았으며, 이는 실제 데이터의 복잡한 변수 간 관계를 합성 데이터가 충분히 포착하지 못함을 보여준다.

4. **효과 크기**: 합성 데이터의 ACWR/Monotony 계수가 실제 데이터보다 수 배~수십 배 큰 것은, 참값 파라미터를 파이프라인 검증에 용이하도록 크게 설정하였기 때문이다. 실제 스포츠 현장에서는 효과 크기가 훨씬 작으며, 임상적/실무적 유의성을 판단할 때 주의가 필요하다.

5. **랜덤 절편 분산**: 합성 데이터(RE var = 3.10~3.12)와 실제 데이터(RE var = 1.57~1.58) 모두에서 상당한 선수 간 변동이 관찰되었으며, ICC가 높은 수준으로 나타나 랜덤 절편의 필요성이 양쪽에서 모두 확인되었다.

## 7. 한계 및 향후 과제

### 7.1 현재 분석의 한계

| 항목 | 설명 |
|------|------|
| **합성 데이터** | 알려진 참값으로 생성된 데이터이므로, 실제 스포츠과학적 해석에 직접 적용할 수 없다 |
| **선형 생성 모형** | 실제 ACWR-Hooper 관계는 비선형(U자형 등)일 수 있다 (Gabbett, 2016) |
| **역인과 미반영** | 선수의 컨디션 상태가 훈련 부하 결정에 영향을 미치는 역인과 관계가 합성 데이터에 포함되지 않았다 |
| **시간적 자기상관** | 잔차의 시간적 자기상관을 모형에 포함하지 않았다 |
| **Strain 미포함** | 실제 데이터 M4에서 유의했던 Strain이 합성 데이터 모형에 미포함이다 |
| **ACWR 낮은 변동성** | ACWR의 SD(0.065)가 매우 작아 계수 추정의 불확실성이 높다 |
| **교란변수 미포함** | 경기일, 이동, 부상 이력 등 실제 현장의 교란변수가 미포함이다 |

### 7.2 향후 과제

1. **Strain 포함 모형 추가**: M5로서 `ACWR + Monotony + Strain + (1|athlete)` 모형을 추가하여 실제 데이터의 억제변수 효과를 재현할 수 있는지 검토한다.
2. **비선형 생성 모형**: ACWR과 Hooper 간 U자형 관계를 반영한 합성 데이터를 생성하여 GAM 등 비선형 모형의 검출력을 평가한다.
3. **역인과 시뮬레이션**: 선수 컨디션 → 부하 결정 → 웰니스 반응의 양방향 인과를 반영한 합성 데이터를 설계한다.
4. **AR(1) 상관 구조**: 시간적 자기상관을 포함한 잔차 구조를 도입하여 모형의 적절성을 향상시킨다.
5. **랜덤 기울기**: `(1 + ACWR|athlete)` 구조로 확장하여 개인별 ACWR 반응 이질성을 분석한다.
6. **교차검증**: LOSO(Leave-One-Subject-Out) CV를 합성 데이터에서도 수행하여 모형의 일반화 가능성을 평가한다.
7. **실제 데이터 재현**: 동일 파이프라인으로 SoccerMon 등 실제 데이터를 분석하여 합성 데이터 결과와 교차 검증한다.
